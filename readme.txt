== Description ==

Plugin automatycznie rozpoznaje postaci w postach i linkuje je do Polskiej Wiki Technologicznej Polishwords.
Wiki znajduje się pod adresem http://polishwords.wikidot.com/.

Aktualna wersja powoduje, że kiedy wpiszesz w treści posta nazwisko osoby z listy http://polishwords.wikidot.com/znani-polacy, to zostanie
ona automatycznie zlinkowana do wpisu bibliograficznego na Wiki.

English

Plugin automatically recognizes persons in posts and links them to Polish TechnoWiki Polishwords.
Wiki is availabile at http://polishwords.wikidot.com/.

== Installation ==

1. Wrzuć `wp-technowiki.php` do foldera `/wp-content/plugins/`
2. Aktywuj wtyczkę w zakładce "Wtyczki"
3. Koniec

English

1. Put 'wp-technowiki.php' to folder '/wp-content/plugins/'
2. Activate plugin in "Plugins" menu
3. End

== Najczęściej zadawane pytania (FAQ) ==

= Czy coś jeszcze trzeba zrobić? =

Nie, wtyczka zadziała od razu. Jeżeli masz system keszowania to trzeba go wyłączyć i włączyć, aby wtyczka mogła przetworzyć posty.

English:

= Is there something more to do? =

Not, plugin works from this moment on. If you have cache plugin you have to switch it off and on again, so that this plugin can process posts.
